## C# Lists (`List<T>`)

A **List** is a dynamic collection. Unlike arrays (`[]`), which have a fixed size, a List **grows and shrinks automatically** as you add or remove elements.

It lives in the `System.Collections.Generic` namespace.

### 1. Declaration & Initialization

You must specify the **type** of data the list will hold using angle brackets `<Type>`.

C#

```C#
using System.Collections.Generic; // Required namespace

// 1. Declare an empty list (Size is 0)
List<int> numbers = new List<int>();

// 2. Initialize with values (Collection Initializer)
List<string> names = new List<string> { "Ana", "Tom", "Zac" };
```

### 2. Adding & Reading Items

C#

```C#
List<int> scores = new List<int>();

// Adding items (.Add)
scores.Add(10);
scores.Add(20);

// Reading items (using Index)
Console.WriteLine(scores[0]); // Output: 10

// Iterating
foreach (int score in scores)
{
    Console.WriteLine(score);
}

// Checking size
// IMPORTANT: Use .Count for Lists (Array uses .Length)
Console.WriteLine(scores.Count); 
```

### 3. Removing Items

You can remove items by their **value** or by their **index**.

C#

```C#
List<string> fruits = new List<string> { "Apple", "Banana", "Orange", "Banana" };

// 1. Remove by Value (.Remove)
// Removes ONLY the FIRST occurrence found.
// Returns true if successful, false if item wasn't found.
bool removed = fruits.Remove("Banana"); 
// List is now: "Apple", "Orange", "Banana"

// 2. Remove by Index (.RemoveAt)
// Removes the item at specific index
fruits.RemoveAt(0); 
// List is now: "Orange", "Banana"

// 3. Clear everything
fruits.Clear(); // List is empty
```

### 4. Sorting (`.Sort`)

The `.Sort()` method reorganizes the elements inside the **original** list.

C#

```C#
List<int> numbers = new List<int> { 5, 1, 9, 3 };

// Sorts in Ascending order (default)
numbers.Sort(); 
// numbers is now: 1, 3, 5, 9

// To reverse it (Descending)
numbers.Reverse();
// numbers is now: 9, 5, 3, 1
```

### 5. Searching (`.FindAll`)

`FindAll` is a powerful method that allows you to filter a list based on a **condition** (Predicate). It returns a **new List** containing all matches.

It uses a **Lambda Expression** (`parameter => condition`).

C#

```C#
List<int> numbers = new List<int> { 10, 5, 20, 8, 30 };

// "Find all numbers 'x' such that 'x' is greater than 10"
// x => x > 10 is the Lambda Expression
List<int> bigNumbers = numbers.FindAll(x => x > 10);

// bigNumbers contains: { 20, 30 }
// The original 'numbers' list is unchanged.

// --- Example with Strings ---
List<string> names = new List<string> { "Ana", "Bob", "Alice", "Tom" };

// Find names starting with 'A'
var aNames = names.FindAll(name => name.StartsWith("A"));
// aNames contains: { "Ana", "Alice" }
```

### Array vs List Cheat Sheet

|**Feature**|**Array (int[])**|**List (List<int>)**|
|---|---|---|
|**Size**|Fixed (cannot change)|Dynamic (grows/shrinks)|
|**Property**|`.Length`|`.Count`|
|**Speed**|Slightly faster|Slightly slower (overhead)|
|**Use when**|You know the exact count.|You don't know how many items.|
