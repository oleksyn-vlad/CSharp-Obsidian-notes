## C# Writing Text Files (`System.IO`)

To read or write files in C#, you must use the System.IO namespace.

This is commonly used for Logging (saving error messages or game history to a .txt file).

**Namespace:** `using System.IO;`

### 1. Two Main Methods

The `File` class provides two static methods to write text easily.

#### A. `WriteAllText` (Overwrite)

Creates a new file. If the file already exists, it **deletes the old one** and replaces it.

- **Use for:** Saving game state, exporting a report.
    

C#

```C#
string path = "log.txt";
string content = "Game Saved: Level 1 complete.";

// WARNING: Replaces everything in 'log.txt'
File.WriteAllText(path, content);
```

#### B. `AppendAllText` (Add to End)

Adds text to the end of the file. If the file doesn't exist, it creates it.

- **Use for:** Logs, History, Journals.
    

C#

```C#
string path = "log.txt";
string newLog = "Player died at 10:05 PM \n"; // \n for new line

// Adds this line to the bottom of the file
File.AppendAllText(path, newLog);
```

### 2. File Paths

- **Relative Path** (`"log.txt"`): Saves the file in the folder where your `.exe` runs (usually `bin\Debug\netX.0\`).
    
- **Absolute Path** (`@"C:\Users\Tom\Desktop\log.txt"`): Saves it exactly where you say.
    

**Tip:** Use the `@` symbol for paths to ignore escape characters like `\`.

C#

```C#
// Good
string path = @"C:\MyLogs\error.txt";

// Bad (Compiler thinks \M is a special code)
string path = "C:\MyLogs\error.txt"; 
```

### 3. Practical Example: Logger Method

This is a simple helper method you can copy into your projects.

C#

```C#
public static void LogError(string message)
{
    string path = "error_log.txt";
    
    // Add timestamp so we know WHEN it happened
    string finalMessage = $"[{DateTime.Now}] {message}\n";
    
    File.AppendAllText(path, finalMessage);
}

// Usage
LogError("Database connection failed.");
LogError("User 'Tom' could not login.");
```
