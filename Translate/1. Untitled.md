
| **English Topic**                  | **Український переклад**               | **Пояснення (Суть теми)**                                                                                              |
| ---------------------------------- | -------------------------------------- | ---------------------------------------------------------------------------------------------------------------------- |
| **Top Level Statements**           | Інструкції верхнього рівня             | Спрощений стиль написання коду без `class Program` і `Main`.                                                           |
| **Using System Directives**        | Директиви `using`                      | Підключення просторів імен (бібліотек), щоб не писати повні шляхи.                                                     |
| **Comments**                       | Коментарі                              | Нотатки в коді, які ігнорує компілятор (`//` або `/* */`).                                                             |
| **Variables & Data Types**         | Змінні та Типи даних                   | Контейнери для даних (`int`, `string` тощо).                                                                           |
| **Naming Conventions**             | Угоди про іменування                   | Правила назв: `camelCase` для змінних, `PascalCase` для класів/методів.                                                |
| **Value vs Reference Type**        | Значущі та Посилальні типи             | `struct` (копіює значення) проти `class` (копіює посилання на пам'ять).                                                |
| **String Interpolation**           | Інтерполяція рядків                    | Вставка змінних у рядок через знак долара: `$"Name: {name}"`.                                                          |
| **String Concatenation**           | Конкатенація рядків                    | З'єднання рядків через плюс: `"Hello " + name`.                                                                        |
| **Escape Characters**              | Екрановані символи                     | Спецсимволи в тексті: `\n` (новий рядок), `\t` (відступ), `\"` (лапки).                                                |
| **Parse / Convert**                | Парсинг / Конвертація                  | Перетворення рядка в число (напр. `int.Parse("10")`).                                                                  |
| **Implicit / Explicit Conversion** | Неявне / Явне перетворення             | Автоматичне (`int`→`double`) або примусове (`(int)myDouble`) змінення типу.                                            |
| **Try Parse**                      | Безпечний парсинг (`TryParse`)         | Спроба перетворити рядок у число без помилки (повертає `bool`).                                                        |
| **Arithmetic Operators**           | Арифметичні оператори                  | Математика: `+`, `-`, `*`, `/`, `%` (остача від ділення).                                                              |
| **Logical Operators**              | Логічні оператори                      | Логіка: `&&` (І), `\|` (АБО), `!` (НЕ).                                                                                |
| **Relational Operators**           | Оператори відношення                   | Порівняння: `==`, `!=`, `>`, `<`, `>=`, `<=`.                                                                          |
| **If / Else / Else If**            | Умовні конструкції                     | Виконання коду тільки при певній умові.                                                                                |
| **Switch Statement**               | Конструкція `switch`                   | Зручний вибір одного варіанту з багатьох (заміна багатьом `if`).                                                       |
| **Variable Scope**                 | Область видимості                      | Де змінна доступна (напр., змінна всередині `{}` невидима ззовні).                                                     |
| **Increment / Decrement**          | Інкремент / Декремент                  | `i++` (збільшити на 1), `i--` (зменшити на 1).                                                                         |
| **For Loop**                       | Цикл `for`                             | Цикл з лічильником (виконується задану кількість разів).                                                               |
| **While Loop**                     | Цикл `while`                           | Виконується, поки умова істинна.                                                                                       |
| **Do While Loop**                  | Цикл `do-while`                        | Виконується мінімум 1 раз, далі — поки умова істинна.                                                                  |
| **Foreach Loop**                   | Цикл `foreach`                         | Перебір усіх елементів колекції/масиву по черзі.                                                                       |
| **Break / Continue**               | Break / Continue                       | `break` (вихід з циклу), `continue` (пропуск поточної ітерації).                                                       |
| **Arrays (Simple)**                | Одновимірні масиви                     | Простий список елементів одного типу `type[]`.                                                                         |
| **Multi-dimensional Arrays**       | Багатовимірні масиви                   | Матриці або таблиці `type[,]`.                                                                                         |
| **Jagged Arrays**                  | Зубчасті масиви                        | Масив масивів `type[][]` (рядки різної довжини).                                                                       |
| **Methods (Void)**                 | Void-методи                            | Функції, які виконують дію, але нічого не повертають.                                                                  |
| **Parameters / Arguments**         | Параметри та Аргументи                 | Дані, які передаються всередину методу.                                                                                |
| **Parameter Modifiers (ref/out)**  | Модифікатори `ref`, `out`              | Передача посилань у метод (зміна змінної всередині міняє її ззовні).                                                   |
| **Recursion**                      | Рекурсія                               | Коли метод викликає сам себе.                                                                                          |
| **Classes & Objects**              | Класи та Об'єкти                       | Клас — шаблон (креслення), Об'єкт — конкретний екземпляр.                                                              |
| **Fields**                         | Поля класу                             | Змінні, що живуть всередині класу (стан об'єкта).                                                                      |
| **Constructors**                   | Конструктори                           | Методи, що запускаються при створенні об'єкта (`new`).                                                                 |
| **Properties**                     | Властивості (`Properties`)             | "Розумні поля" з `get` та `set` для контролю доступу до даних.                                                         |
| **Computed Properties**            | Обчислювані властивості                | Властивості, які не зберігають дані, а рахують їх при зверненні (тільки `=>`).                                         |
| **Expression Bodied Members**      | Члени-вирази (`=>`)                    | Скорочений синтаксис методів/властивостей через стрілку.                                                               |
| **Partial Classes**                | Часткові класи                         | Розділення коду одного класу на кілька файлів.                                                                         |
| **Inner (Nested) Classes**         | Вкладені класи                         | Клас, оголошений всередині іншого класу.                                                                               |
| **Optional Parameters**            | Необов'язкові параметри                | Параметри зі значенням за замовчуванням (`int x = 0`).                                                                 |
| **Named Parameters**               | Іменовані параметри                    | Передача аргументів за іменем (`Func(age: 20)`), порядок не важливий.                                                  |
| **Operator Overloading**           | Перевантаження операторів              | Навчання програми використовувати `+`, `-`, `==` для твоїх класів.                                                     |
| **Passing by Value vs Reference**  | Передача за значенням та за посиланням | ref, out                                                                                                               |
| **Computed Properties**            | Обчислювані властивості                | Властивості, які не зберігають дані в пам'яті, а обчислюють результат "на льоту" (мають тільки логіку `get` або `=>`). |
