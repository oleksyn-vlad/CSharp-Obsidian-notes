
## C# Arrays (Simple)

Arrays in C# are objects that represent a fixed-size collection of elements of the same type. The syntax is slightly different from C++.

### 1. Declaration and Initialization

The square brackets `[]` come **after the type**, not the variable name.

C#

```C#
// 1. Declare and initialize with size (default values)
// Elements are initialized to their default (0 for int, null for objects)
int[] numbers = new int[5]; // Creates an array of 5 integers (all are 0)

// 2. Declare and initialize with values (shorthand)
// The size is inferred from the number of elements
string[] names = { "Ana", "Tom", "Zac" };

// 3. Alternative (more explicit) initialization
double[] grades = new double[] { 95.5, 80.0, 99.1 };

// 4. Declaring without initializing
int[] myUninitializedArray;
// You MUST initialize it before use
myUninitializedArray = new int[10];
```

### 2. Accessing Elements

Accessing elements is identical to C++, using zero-based indexing.

C#

```C#
string[] names = { "Ana", "Tom", "Zac" };

// Access the first element
Console.WriteLine(names[0]); // Output: Ana

// Modify an element
names[1] = "Thomas";
Console.WriteLine(names[1]); // Output: Thomas
```

### 3. Key Properties: .Length

Unlike C/C++, C# arrays have a built-in `Length` property (note: **it's a property, not a method**, so no parentheses `()`).

C#

```C#
int[] numbers = { 10, 20, 30, 40 };
Console.WriteLine(numbers.Length); // Output: 4
```

### 4. Iteration

You can use a standard `for` loop or the simpler `foreach` loop.

C#

```C#
int[] scores = { 100, 88, 92 };

// 1. Using a standard 'for' loop (useful if you need the index)
for (int i = 0; i < scores.Length; i++)
{
    Console.WriteLine($"Index {i}: {scores[i]}");
}

// 2. Using 'foreach' (simpler, read-only iteration)
// This is the preferred way when you just need the values
foreach (int score in scores)
{
    Console.WriteLine(score);
}
```
