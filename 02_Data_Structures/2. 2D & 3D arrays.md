## C# Multidimensional Arrays

C# supports two types of multi-dimensional arrays:

1. **Rectangular Arrays (or "True" MD Arrays):** A single object with multiple dimensions (e.g., `int[,]`).
    
2. **Jagged Arrays:** An array _of arrays_ (e.g., `int[][]`).
    

This note focuses on **Rectangular Arrays**, which are simpler to use for fixed grid/matrix structures.

### 1. Two-Dimensional (2D) Arrays

A 2D array is a matrix (like a grid or spreadsheet). It has rows and columns.

**Syntax:** `type[ , ]` (a comma separates the dimensions).

#### Declaration & Initialization

C#

```C#
// 1. Declare and initialize with size (default values)
// Creates a 3x4 grid (3 rows, 4 columns)
int[,] matrix = new int[3, 4];

// 2. Declare and initialize with values
string[,] grid = 
{
    { "A1", "B1", "C1" },
    { "A2", "B2", "C2" }
};
// This creates a 2x3 grid
```

#### Accessing Elements & Dimensions

Access is `array[row, col]`. Use `.GetLength(dim)` to find the size of a specific dimension.

C#

```C#
int[,] matrix = new int[3, 4];

// Assign a value
matrix[0, 1] = 5; // Row 0, Column 1

// Get dimensions
int rows = matrix.GetLength(0); // Dimension 0 is rows (3)
int cols = matrix.GetLength(1); // Dimension 1 is columns (4)

Console.WriteLine($"Grid is {rows}x{cols}");
```

#### Iterating a 2D Array

You must use nested `for` loops. `foreach` will flatten the array, which is usually not what you want.

C#

```C#
string[,] grid = 
{
    { "A1", "B1" },
    { "A2", "B2" },
    { "A3", "B3" }
};

// Standard iteration
for (int row = 0; row < grid.GetLength(0); row++)
{
    for (int col = 0; col < grid.GetLength(1); col++)
    {
        Console.Write(grid[row, col] + " "); // Prints "A1 B1 A2 B2 A3 B3"
    }
    Console.WriteLine(); // Newline after each row
}
```

### 2. Three-Dimensional (3D) Arrays

This extends the same concept to a 3rd dimension (like a cube).

**Syntax:** `type[ , , ]`

C#

```C#
// 1. Declare and initialize (e.g., a 2x3x4 "cube")
double[,,] dataCube = new double[2, 3, 4];

// 2. Initialize with values
int[,,] cube = 
{
    { { 1, 2 }, { 3, 4 } }, // Layer 0
    { { 5, 6 }, { 7, 8 } }  // Layer 1
};
// This is a 2x2x2 cube

// 3. Accessing
// cube[layer, row, column]
Console.WriteLine(cube[1, 0, 1]); // Output: 6

// 4. Getting dimensions
int dim1 = cube.GetLength(0); // 2
int dim2 = cube.GetLength(1); // 2
int dim3 = cube.GetLength(2); // 2

// 5. Iterating (requires 3 nested loops)
for (int i = 0; i < cube.GetLength(0); i++)
{
    for (int j = 0; j < cube.GetLength(1); j++)
    {
        for (int k = 0; k < cube.GetLength(2); k++)
        {
            // Process cube[i, j, k]
        }
    }
}
```
