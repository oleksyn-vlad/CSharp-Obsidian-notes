## C# foreach Loop

The `foreach` loop is the simplest and most reliable way to iterate (loop over) all elements in a collection, such as an `array` or a `List`.

It replaces the standard `for (int i = 0; i < collection.Length; i++)` loop when you **do not need the index** and only care about the value of each element.

### 1. Syntax

C#

```C#
foreach (var item in collection)
{
    // Your code that uses 'item'
}
```

- `var`: The `var` keyword automatically infers the data type of the element (e.g., `int`, `string`, etc.).
    
- `item`: This is a temporary, local variable that holds a _copy_ of the current element on each iteration.
    
- `in`: The keyword that separates the variable from the collection.
    
- `collection`: The array, list, or any other object that can be iterated.
    

### 2. Examples

#### Example 1: Iterating an `int[]` array

C#

```C#
int[] numbers = { 1, 5, 10, 15 };
int sum = 0;

foreach (int number in numbers)
{
    Console.WriteLine($"Current number: {number}");
    sum += number;
}
// 'sum' is now 31
```

_(You could also write `foreach (var number in numbers)`)_

#### Example 2: Iterating a `List<string>`

C#

```C#
List<string> names = new List<string> { "Ana", "Tom", "Zac" };

foreach (var name in names)
{
    // Convert the name to uppercase
    Console.WriteLine(name.ToUpper());
}
// Output:
// ANA
// TOM
// ZAC
```

### 3. Important Limitation: Read-Only

**You cannot modify the collection while using `foreach`**.

The iteration variable (`item`) is effectively **read-only**. You cannot use it to assign a new value back into the original collection.

C#

```C#
int[] numbers = { 1, 2, 3 };

// THIS CODE WILL NOT CHANGE THE ARRAY
foreach (var num in numbers)
{
    // num = num * 2; // This only changes the local copy 'num',
                    // not the element in the 'numbers' array.
}
```

> **Conclusion:** `foreach` is ideal for **reading** data from a collection. If you need to **modify** the elements in the collection, use a standard `for` loop with an index (e.g., `for (int i=0; ...)`).
