## C# String Methods: ToLower & Trim

A key concept in C# (different from C++) is that **strings are immutable**.

> Methods like `ToLower()` or `Trim()` **do not modify the original string**. They **return a new string** with the applied changes. You must assign this new string to a variable.

### 1. ToLower() & ToUpper()

These methods convert the case of the entire string.

C#

```C#
// .ToLower() returns a new string in all lowercase
string original = "Hello, C# World!";
string lower = original.ToLower();

// .ToUpper() returns a new string in all uppercase
string upper = original.ToUpper();

// The original string is unchanged:
Console.WriteLine(original); // Output: Hello, C# World!

Console.WriteLine(lower);    // Output: hello, c# world!
Console.WriteLine(upper);    // Output: HELLO, C# WORLD!
```

### 2. Trim()

This method removes whitespace (spaces, tabs, newlines) from the **beginning and end** of a string.

C#

```C#
// 1. Standard Trim
// Note the newlines (\n) and spaces
string messyInput = "   \n  Hello   ";
string clean = messyInput.Trim();

// We use quotes to visualize the removed whitespace
Console.WriteLine($"Original: '{messyInput}'");
Console.WriteLine($"Clean:    '{clean}'"); // Output: 'Hello'


// 2. TrimStart() and TrimEnd()
// You can trim only one side
string trimStart = messyInput.TrimStart(); // "'Hello   '"
string trimEnd = messyInput.TrimEnd();     // "'   \n  Hello'"


// 3. Trim(char[])
// You can also trim specific characters, not just whitespace
string withChars = "***MyData###";
char[] charsToTrim = { '*', '#' };

string trimmedChars = withChars.Trim(charsToTrim);
Console.WriteLine(trimmedChars); // Output: MyData
```
