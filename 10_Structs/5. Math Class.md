## C# Math Class (`System.Math`)

The `Math` class is a built-in **Static Class** that provides constants and static methods for common mathematical functions.

**Key Rule:** Because it is static, you **never** create an object of it (`new Math()` is impossible). You call the methods directly on the class name.

### 1. Basic Constants

C#

```C#
// PI (3.14159...)
double area = Math.PI * radius * radius;

// E (2.718...)
double e = Math.E;
```

### 2. Rounding Methods

C# offers three distinct ways to handle floating-point numbers.

C#

```C#
double num = 9.5;

// 1. Ceiling (Always rounds UP)
Console.WriteLine(Math.Ceiling(num)); // 10

// 2. Floor (Always rounds DOWN)
Console.WriteLine(Math.Floor(num));   // 9

// 3. Round (Nearest Integer)
// Note: C# uses "Banker's Rounding" (rounds to nearest EVEN number on .5)
Console.WriteLine(Math.Round(9.5));   // 10
Console.WriteLine(Math.Round(8.5));   // 8
```

### 3. Powers & Roots

C#

```C#
// Power: 2 to the power of 3 (2 * 2 * 2)
double power = Math.Pow(2, 3); // 8

// Square Root
double root = Math.Sqrt(64);   // 8
```

### 4. Comparison & Absolute Value

C#

```C#
int a = -10;
int b = 20;

// Absolute (Positive value)
Console.WriteLine(Math.Abs(a)); // 10

// Min & Max (Compare two numbers)
Console.WriteLine(Math.Max(a, b)); // 20
Console.WriteLine(Math.Min(a, b)); // -10
```

### 5. Clamping (Newer C#)

If you need to keep a value between a min and max (common in games for health bars), use `Math.Clamp`.

C#

```C#
// Ensure health stays between 0 and 100
int health = 150;
int result = Math.Clamp(health, 0, 100); // Returns 100
```
